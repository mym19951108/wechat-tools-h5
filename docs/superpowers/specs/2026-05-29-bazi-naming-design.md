# 八字取名功能设计文档

## 概述

改造现有宝宝取名工具，新增八字取名模式。用户输入姓氏、性别、出生日期时间，系统根据生辰八字分析五行缺失+喜忌，从字库筛选匹配字，并从诗词典籍中溯源每个候选名用字的出处。零后端，纯前端计算。

## 架构

```
BabyName.vue (改造)
├── Tab "智能取名" → BabyNameEngine.js (不变)
└── Tab "八字取名"
    ├── 输入: 姓氏 + 性别 + 出生日期 + 时辰
    ├── BaziEngine.js (新) → lunar-javascript → 八字四柱 + 五行统计 + 喜忌分析
    ├── BaziNameEngine.js (新) → names.js → 五行喜忌匹配 + 性别过滤 → 候选名
    ├── poetry-map.js (新, lazy import) → 诗词溯源
    └── 展示: 八字排盘 + 五行分析 + 喜忌建议 + 候选名(含诗词出处)
```

## 技术选型

| 项 | 选型 | 理由 |
|---|---|---|
| 农历/八字库 | `lunar-javascript` ^1.6 | 成熟库，自带八字四柱、五行 |
| 诗词数据库 | 静态 JS Map 文件，动态 import | Vite 自动拆 chunk，不增加首屏体积 |
| 加载策略 | 用户点"生成名字"时触发 import | 首屏零额外体积 |
| 计算位置 | 纯前端 | 零后端 |

## 数据流

```
用户点击"分析八字取名"
  │
  ├─→ loading = true
  │
  ├─→ BaziEngine.analyze(year, month, day, hour)
  │   ├── Solar.fromYmdHms() → Lunar → EightChar
  │   ├── 四柱: { yearPillar, monthPillar, dayPillar, hourPillar }
  │   ├── 五行统计: { 金:n, 木:n, 水:n, 火:n, 土:n }
  │   ├── 缺失: [missing elements]
  │   ├── 日主天干: dayPillar.stem
  │   ├── 喜忌分析 (扶抑法):
  │   │   ├── 日主五行属性
  │   │   ├── 出生季节 → 判断身强/身弱
  │   │   ├── 喜用神 (宜补的五行) ← 身弱喜生扶, 身强喜克泄耗
  │   │   └── 忌神 (应避的五行)
  │   └── 输出 → BaziNameEngine
  │
  ├─→ BaziNameEngine.generate({ surname, gender, xiShen, jiShen })
  │   ├── 从 names.js 筛选喜用神五行的字 (优先)
  │   ├── 从 names.js 筛选补缺五行的字 (次选)
  │   ├── 排除忌神五行的字
  │   ├── 按性别过滤字池
  │   ├── 组合: 姓氏 + 喜用字 + 补缺字
  │   ├── 打分: 喜忌匹配度 + 音韵 + 寓意
  │   └── 输出: 20 个候选名
  │
  ├─→ const poetryMap = await import('../../data/poetry-map.js')  ← lazy load
  │   ├── 每个候选名的用字查诗词映射
  │   └── 附上包含该字的诗句 (最多2句/字)
  │
  ├─→ loading = false
  │
  └─→ UI 展示: 八字排盘 + 喜忌分析 + 候选名(含诗词出处)
```

## 五行喜忌算法（简化版扶抑法）

```
1. 取日柱天干，映射五行：
   甲乙=木 丙丁=火 戊己=土 庚辛=金 壬癸=水

2. 判断身强/身弱（月令法）：
   - 日主五行在出生季节得令 → 身强
   - 日主五行在出生季节失令 → 身弱

3. 喜忌规则：
   - 身强 → 喜克泄耗 (克:克日主的五行, 泄:日主生的五行, 耗:生克日主的)
   - 身弱 → 喜生扶 (生:生日主的五行, 扶:同日主的五行)

4. 输出：喜用神五行列表 (xiShen[]), 忌神五行列表 (jiShen[])
```

## 诗词溯源

**数据来源：**
- 诗经、楚辞、唐诗三百首、宋词三百首、古诗十九首
- 全量，约 2000+ 首，按字索引

**数据结构：**
```js

  '涵': [
    { text: '海纳百川，有容乃大', source: '《尚书》' },
    { text: '涵虚混太清', source: '孟浩然《望洞庭湖赠张丞相》' }
  ],
  '宇': [
    { text: '宇泰定者，发乎天光', source: '《庄子》' },
    { text: '气宇轩昂出世间', source: '...' }
  ]
  // ...

```

**加载策略：**
- 用户点"分析八字取名"时 `await import('../../data/poetry-map.js')`
- Vite 自动拆为独立 chunk（预计 150-200KB gzipped ~50KB）
- 加载时显示 loading spinner: "正在分析八字，请稍候..."
- 如某字在诗词库中未匹配，显示"暂无典籍记录"

## 生成逻辑

1. 八字计算 → 四柱 + 五行统计
2. 喜忌分析 → 喜用神 + 忌神
3. 字池筛选：喜用神五行字优先，补缺字次选，排除忌神字
4. 按性别过滤
5. 组合打分：喜忌匹配度(40%) + 音韵(30%) + 寓意(30%)
6. 动态加载诗词库，匹配用字出处
7. 输出 20 个候选名

## 测试用例

已知八字：1990-05-20 12:00

- 年柱: 庚午, 月柱: 辛巳, 日柱: 乙酉, 时柱: 壬午
- 五行: 金2 木1 水1 火3 土0 → 缺土
- 日主: 乙木 → 巳月(夏) → 木在夏季失令 → 身弱
- 身弱 → 喜生扶 → 喜用神: 水(生木) 木(扶木)
- 忌神: 金(克木) 火(木生火泄气)
- 推荐: 含水/木五行的字
