# 微信公众号工具号 H5 设计文档

## 概述

从零搭建微信公众号变现项目。纯静态 H5 工具页面，公众号菜单引流，流量主广告变现。首批上线 5 个工具，后续按数据迭代。

## 架构

纯静态 SPA，Vue 3 + Vite 构建，CloudFlare Pages 托管。零后端、零数据库起步。

```
微信公众号菜单/关键词
        │
        ▼
微信内置浏览器 → CloudFlare Pages (CDN)
        │
        ▼
    Vue 3 SPA
   ├── 工具页面（纯前端运算）
   └── 广告位（微信流量主 JS SDK）
```

## 技术栈

| 层 | 选型 | 理由 |
|---|---|---|
| 前端框架 | Vue 3 + Composition API | 轻量，响应式，生态成熟 |
| 构建工具 | Vite | 打包快，开箱支持移动端适配 |
| UI 库 | 自写 CSS（移动端 rem 适配） | 体积小，完全控制 |
| 静态托管 | CloudFlare Pages | 免费额度充足，全球 CDN |
| 版本管理 | GitHub | 推送自动触发部署 |

## 首批工具

### 起名类

1. **宝宝取名** — 输入姓氏、性别、偏好（五行/寓意），生成 20+ 候选名，含寓意解释和打分
2. **英文名生成** — 输入中文名，匹配音近英文名，含寓意和来源说明
3. **网名生成** — 选择风格（古风/简约/可爱/霸气），随机组合生成 10+ 网名

### 计算类

4. **房贷计算** — 输入总额、利率、年限，输出等额本息/等额本金月供及还款计划表
5. **日期计算** — 日期间隔天数、日期推算（N天后是几号）、工作日计算

## 广告策略

- 500 粉前：广告位显示"关注公众号"引导图，不浪费位置
- 500 粉后：替换为微信流量主广告代码
- 广告位使用 AdSlot 组件统一管理，一键切换

## 测试策略

| 层 | 内容 | 工具 |
|----|------|------|
| 算法 | 取名结果正确性、计算精度 | Vitest |
| UI | 移动端适配、不同机型兼容 | 微信开发者工具 + 真机 |
| 性能 | 首屏 < 2s、包体积 < 200KB | Vite build + Lighthouse |
| 集成 | 菜单跳转、JS 安全域名 | 灰度自测 |

## 冷启动策略

1. 工具上线后，公众号每天推送一篇工具使用教程/场景文章（如"房贷计算怎么用最划算"）
2. 每篇文章底部引导关注，引导用工具
3. 工具页面底部互相导流，拉动多工具使用

## 不做什么（YAGNI）

- 不做用户登录系统
- 不做后端 API
- 不做数据库存储
- 不做付费解锁
- 不做数据统计后台（用 CloudFlare Analytics 看基础 PV）

## 成功指标

- 第一阶段目标：500 粉，开通流量主
- 第二阶段目标：日均 PV 1000+，月广告收入 ¥200+
